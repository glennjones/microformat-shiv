microformat-shiv

A light-weight cross browser javascript microformats 2 parser

h-adr

Below is an example of the microformat-shiv parsing a h-adr. This microformat is used to describe addresses and is based on the vCard standard used in most email and office software.

Example h-adr

Suite 300, 650 Castro Street, Mountain View, CA 94041, United States test.html

Specifications compliance

The library supports both version 1 and 2 microfomat HTML mark-up patterns. You can check the compliance of the microformat-shiv to the h-adr standard by using the microformats test suite.
